依頼に基づいて、サイトの `/recommend` ページで表示される「更新日不明」を最終更新日に変更するためのコードを提案します。ここでは、サイトがHTMLとJavaScriptを使用していると仮定しています。また、サーバーサイドでの日付の取得と送信が可能であることを前提とします。 ### 1. サーバーサイドの変更（例: Node.jsを使用） サーバーサイドでファイルの最終更新日を取得し、クライアントに送信するコードを追加します。 ```javascript const express = require('express'); const fs = require('fs'); const path = require('path'); const app = express(); const port = 3000; app.get('/recommend', (req, res) => { const filePath = path.join(__dirname, 'path/to/recommend.html'); fs.stat(filePath, (err, stats) => { if (err) { return res.status(500).send('Error retrieving file info'); } res.send({ content: fs.readFileSync(filePath, 'utf8'), lastModified: stats.mtime.toISOString().split('T')[0] // YYYY-MM-DD format }); }); }); app.listen(port, () => { console.log(`Server running on port ${port}`); }); ``` ### 2. クライアントサイドのJavaScript変更 クライアントサイドで受け取った最終更新日を表示するためのJavaScriptコードを追加します。 ```html Recommend Page
Last updated:
``` ### 説明 1. **サーバーサイド**: Node.jsを使用して、指定されたファイルの最終更新日時を取得し、それをクライアントに送信します。`fs.stat`を使用してファイルのメタデータを取得し、その中の`mtime`プロパティから最終更新時刻を取得しています。 2. **クライアントサイド**: JavaScriptの`fetch` APIを使用してサーバーからページ内容と最終更新日を取得します。取得したデータはHTMLに適切に表示されます。 この変更により、`/recommend`ページの「更新日不明」が実際の最終更新日に更新され、ユーザーに対してより正確な情報が提供されます。
